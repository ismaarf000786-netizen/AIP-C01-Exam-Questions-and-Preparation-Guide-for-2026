## Exam Domains and Weighting
The exam is divided into **five key domains**, each reflecting real production responsibilities:

**1) Foundation Model Integration & Data Management (31%)**  
Focuses on selecting and integrating foundation models (often via **Amazon Bedrock**), handling data ingestion, managing vector databases, and ensuring compliance.

**2) Implementation & Integration (26%)**  
Covers **Retrieval-Augmented Generation (RAG)** architectures, knowledge bases, API orchestration, and end-to-end application workflows.

**3) AI Safety, Security & Governance (20%)**  
Tests responsible AI principles, bias detection, IAM policies, data privacy, and governance controls for GenAI applications.

**4) Operational Efficiency & Optimization (12%)**  
Evaluates cost control, performance tuning, scalability, and monitoring strategies in production environments.

**5) Testing, Validation & Troubleshooting (11%)**  
Focuses on evaluation metrics, debugging model behavior, and resolving real production incidents.
